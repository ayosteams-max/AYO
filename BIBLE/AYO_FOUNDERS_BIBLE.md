# Every Feature Must Answer

Before building anything ask:

1. Does this help customers?
2. Does this help drivers?
3. Is it secure?
4. Is it simple?
5. Is it fast?
6. Is it reliable?
7. Can millions of people use it?
8. Would I personally love using it?

If the answer is NO...

Don't build it.
