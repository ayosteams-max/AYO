# Mission 21 — Risk Register

Status: **Architecture proposal only. No critical risk is accepted by this document.**

| Risk | L/I | Mitigation | Residual owner | Verification |
|---|---|---|---|---|
| Emergency under-escalation | M/Critical | Deterministic fast path, recall-first rules, specialist ownership | Safety/CEO | Native-language red team and drills |
| Hallucinated policy/evidence | H/High | Structured citations, policy renderer, unsupported-claim rejection | CTO | Grounding/citation suite |
| Unfair rider/driver outcomes | M/High | Bilateral evidence, parity metrics, appeal/reopen, no hidden reputation | Product/Ops | Split fairness and override analysis |
| Cross-case disclosure | M/Critical | Scoped gateway, RBAC, masking, isolation tokens | Security/Privacy | IDOR and exfiltration tests |
| AI authority creep | M/Critical | Matrix-enforced tool classes; high-risk mandatory routing | CTO/CEO | Forbidden-action contracts |
| Amharic meaning loss | H/High | Original retained, glossary/version, correctable language, native review | Language Ops | Blind native-speaker evaluation |
| Weak network/provider outage | H/High | Durable case, deterministic fallback, visible stages | CTO/Ops | interruption and outage tests |
| Upload malware/privacy | M/High | Metadata-only architecture, quarantine gate, no model execution | Security/Privacy | malicious corpus and retention review |
| Case backlog hides urgency | M/Critical | deadline policy, aging escalation, staffed queue SLO | Operations | load/soak and staffing drills |
| Retention/cross-border violation | M/Critical | purpose classes, regional/provider gate, legal hold/deletion separation | Legal/Privacy | DPIA and deletion/hold tests |
| Cost/latency spike or lock-in | M/Medium | bounded context/tools, adapter, quotas, fallback, cost telemetry | CTO/Finance | load/cost/provider swap evaluation |
| Premature automation | M/High | shadow then assist; separately approved allow-list | CTO/CEO | stage-gate metrics |
| Visual replay exposes movement | M/Critical | Coarse purpose projection, redaction, expiry, no raw GPS | Privacy/Security | Re-identification and role-leak tests |
| Simple wording changes policy | M/High | Reviewed versioned components and bilingual fidelity tests | Legal/Product | Native-speaker and policy-owner review |
| Voice/video captures bystanders | M/Critical | Consent, visible indicators, recording off, minimised metadata | Privacy/Legal | Consent/revoke scenarios |
| Screen sharing exposes credentials | M/Critical | Field allow-list, masking, no generic control, revoke | Security | Secret-field tests |
| Live location reused beyond purpose | M/Critical | Precision/recipient/expiry grant and audited deletion | Privacy/Safety | Expiry/purpose tests |
| Family representative exceeds authority | M/High | Typed roles and per-action authorization | Identity/Legal | Confused-deputy tests |
| Quality score becomes punishment | M/High | Aggregate purpose and prohibited consequential use | CEO/People Ops | Fairness audits |
| Reviewed data poisons learning | M/High | Eligibility, provenance, sampling audit and holdout | AI Governance | Poisoning/drift tests |
