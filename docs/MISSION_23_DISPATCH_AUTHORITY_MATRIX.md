# Mission 23 — Dispatch Authority Matrix

| Boundary | Owns | Mission 23 may do | Must never do |
|---|---|---|---|
| Immediate Dispatch | Immediate search/offers/assignment | Supply policy inputs/shadow evaluation | Assign independently or add marginal delay |
| Scheduled Engine | Planning, commitment, recovery | Supply health/market context | Break commitment lock |
| Smart Pre-Dispatch | Conditional next-trip offer | Recommend compatible opportunity | Start before current completion |
| Active Ride | Post-assignment lifecycle | Read bounded completion/health | Change trip state |
| Airport Standard/Premium | Separate queue/product policy | Coordinate zone/queue health | Merge promises or calculate charges |
| Mission 20 | Arrival/readiness/wait evidence | Read only after certification | Activate, blame or dispatch |
| Availability | Online lease/location/work constraints | Consume fresh projection | Predict availability as fact |
| Eligibility/Safety | Verification/restriction gates | Apply returned result | Override or expose restricted reasons |
| Pricing/Incentives | Fare/incentive policy | Recommend review | Set prices, bonuses or money |
| Support | Cases/appeals | Receive redacted evidence | Rewrite dispatch history |
| Marketplace Health | Aggregate advisory metrics | Recommend typed action | Dispatch, price or restrict |
| AI/prediction | Estimates with provenance | Advise deterministic policy | Assign/remove/punish/bypass locks |
